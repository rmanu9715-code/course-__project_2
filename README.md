## Problem Statement

The objective of this project is to design and implement a Travel Booking Platform using Linked List data structure in C language. The system allows users to manage travel bookings efficiently through CRUD operations such as adding, updating, deleting, searching, and displaying booking records.


## Objective

To implement a menu-driven Travel Booking Platform using dynamic memory allocation and Linked List concepts in C programming.


## Data Structure Used

Singly Linked List

Each node stores:

Booking ID

Passenger Name

Destination

Number of Seats

Ticket Price


## Features (CRUD Operations)

Create → Add new booking

Read → Display booking records

Update → Modify booking details

Delete → Remove booking record

Search → Find booking using Booking ID


## Algorithm

Step 1: Start program

Step 2: Create structure for booking details

Step 3: Allocate memory dynamically using malloc()

Step 4: Display menu options

Step 5: Perform operations:

Add booking

Delete booking

Update booking

Search booking

Display bookings

Step 6: Exit program
